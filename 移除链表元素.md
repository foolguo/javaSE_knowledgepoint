# 移除链表元素

直接法

```java
   public ListNode removeElements(ListNode head, int val) {
        while(head!=null&&head.val==val){
            head=head.next;
    }
    if(head==null){
            return null;
    }
    ListNode pre=new ListNode(1);
    while(pre.next!=null){
        if(pre.next.val==val){
            pre.next=pre.next.next;

        }else{
            pre=pre.next;
        }
    }
    return head;
}
```

dummyHead法

```java


  //Definition for singly-linked list.
   class ListNode {
      int val;
      ListNode next;
      ListNode(int x) { val = x; }
  }

class Solution {
    /**
     * 1.一个前驱结点，一个当前节点
     * 2.循环内：
     * （1）
     *
     * 首先创建一个虚拟头结点放到头结点的前面，这样每个头结点都会有一个前驱结点，这样的话
     *
     *
     * @param head
     * @param val
     * @return
     */
    //自己的方法：
    public ListNode removeElements(ListNode head, int val) {
        ListNode dummyHead=new ListNode(0);
        dummyHead.next=head;
            for (ListNode prev = dummyHead, curr = head; curr != null; ) {
                ListNode temp = curr.next;
                if (curr.val == val) {
                    prev.next = curr.next;
                    curr.next = null;
                    curr = null;
                    curr = temp;

                } else {
                    prev = prev.next;
                    curr = curr.next;
                }
            }
       return dummyHead.next;
    }
}
```
